## 🔑 Key Differences

| Aspect | Client | Lawyer |
|--------|--------|--------|
| **Add Money** | ✅ Yes | ❌ No |
| **Receives From** | Self | Clients |
| **Main Action** | Add Money | Withdraw |
| **Transaction Details** | Service type | Client name |
| **Focus** | Spending | Earning |
